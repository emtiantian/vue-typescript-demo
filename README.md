# show-v2.0

## Project setup 初始化项目
```
yarn install
```

### Compiles and hot-reloads for development  开发时启用本地服务器
```
yarn run serve
```

### Compiles and minifies for production 部署前打包
```
yarn run build
```

### Lints and fixes files 测试格式是否正确
```
yarn run lint
```

### Run your unit tests  单元测试
```
yarn run test:unit
```

### Run your end-to-end tests 段对端测试 e-to-e
```
yarn run test:e2e
```
